Analog I/O Configuration of the netPLC I/O Handler Facility

This tag configures the mapping between an analogue I/O and a value in the PLC I/O image.

Image Type None, Input or Output
Converter Device Number Range: 0 ... 1
Channel Number Range: 0 ... 3
Number Format The data format to appear in the PLC I/O image
  • 16-bit signed integer
  • 16-bit unsigned integer
  • IEEE 10-bit signed
  • IEEE 10-bit unsigned
  • S7
Image Offset Start offset in the PLC I/O image (in bytes)
