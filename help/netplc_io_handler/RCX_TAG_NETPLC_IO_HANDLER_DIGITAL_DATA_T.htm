Digital I/O Configuration of the netPLC I/O Handler Facility

This tag configures the mapping between a set of consecutive digital I/O pins and a value in the PLC I/O image.

Image Type None, Input or Output
I/O Type PIO, HIF PIO or GPIO
Index of First Bit Index of the first PIO, HIF-PIO or GPIO that starts the set
Number of Bits Number of consecutive PIOs, HIF-PIOs or GPIOs in the set
Range: 1 ... 32
Invert Mask bit=1: invert I/O data bit after reading netX input register / before writing netX output register
Image Offset Start offset in the PLC I/O image (in bytes)
Bit Offset Start offset from the beginning of the first associated byte in the PLC I/O image (in bits)
Range: 0 ... 7
