USB Descriptor Parameters

USB descriptor parameters for netX 10 can be set using this tag. If no parameters are given, the 2nd Stage Loader will use the default settings.

Custom Settings Check to use the custom settings in the remaining fields. Otherwise, the default settings will be used.
Vendor ID USB vendor ID assigned by the USB Implementers Forum, Inc. (word)
Product ID USB product ID assigned by the vendor. (word)
Release Number Device release number. (4 digits, BCD encoding)
Characteristics Configuration characteristics.
  • Self-powered
  • Bus-powered
  • Self-powered, supports Remote Wakeup
  • Bus-powered, supports Remote Wakeup
Maximum Power Maximum power consumption of the USB device from the bus in this specific configuration when the device is fully operational. Expressed in 2 mA units (i.e., 50 = 100 mA).
Vendor Name Name string associated with the vendor ID. (Max. 16 alphanumerical characters.)
Product Name Name string associated with the product ID. (Max. 16 alphanumerical characters.)
Serial Number Serial number string associated with the device. (Max. 16 numerical characters.)
