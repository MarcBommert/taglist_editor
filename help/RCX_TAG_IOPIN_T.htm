IO Pin Definition

This tag defines an IO pin.

Resource TypeThe type of I/O (PIO, GPIO)
Pin NumberThe number of the pin
PolarityThe polarity (normal or inverted)

The function of the pin and its direction (input/output) is defined by the firmware and is indicated by the identifier string:

Identifier Direction Function
Ready Output Indicates if the device is ready
Start/Stop Com Input Tells the device to start or stop communication
