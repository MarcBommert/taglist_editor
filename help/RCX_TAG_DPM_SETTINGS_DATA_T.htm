Redefining DPM Settings

This tag allows the user to modify the RX_HIF_SET_T describing DPM location and access.

DPM Mode8 bit, 16 bit or PCI (default)
DPM SizeThe DPM size in bytes.
The default size is 16384 bytes for comX, 32768 for other targets.
DPM Base AddressThe base address of the DPM in INTRAM. The default is 0x18000.

Note: The values REPLACE the statically defined values at system startup time, i.e. they serve as ABSOLUTE VALUES.

Restrictions

  • DPM Base Address >= 0x80 (The first 128 bytes are used by the exception vector table.)
  • DPM Size >= 0x1200 bytes (512 + 4096 bytes are needed for the system channel and a minimal communication channel.)
  • DPM Size must be a multiple of 256 bytes
  • DPM Base Address + DPM Size <= end of the INTRAM
