Fiber Optic DMI (netX100)

Fiber Optic Transceiver Diagnostic and Monitoring Interface (DMI) settings for netX100 designs. This tag defines the I2C Transceiver Select Pin used to switch between DMI Interfaces of both Transceivers.

I2C Transceiver Select Pin Type Pin type (GPIO/PIO) for Transceiver Select Pin
Pin Number Pin number for Transceiver Select Pin
PIO: 0-84
GPIO: 0-15
Inverted If checked, the polarity of the signal is inverted.
