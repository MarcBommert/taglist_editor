Ethernet/IP EDD Configuration

This tag selects which type of Ethernet/IP EDD functionality and which xC unit(s) are used.

Enable DLR Enables the DLR (Device Level Ring) functionality. If enabled, the xC type must be set to "Ethernet DLR".
xC Type Defines the type of xC to use:
  • Ethernet DLR: Standard 2 port switch with integrated DLR functionality. This XC type uses xC number 0, 1 and 3. It must be selected when DLR is enabled.
  • Ethernet 2 port switch: Standard 2 port switch. This XC type uses xC number 0 and 1.
  • Ethernet single port: Standard single port xC. The XC number can be defined in the field "Single Port xC Number".
xC number If xC type "Ethernet single port" is used, this field defines the XC number (possible values: 0-3). The value will be ignored for other xC types.
