Sercos III Product Information

The following parameters define a Sercos III device identification.

Vendor Code Vendor identification number of the manufacturer, which has been assigned to the vendor by SERCOS International.
This number is the same as defined in Sercos IDN S-0-1300.x.03
All Hilscher products use the value 1000 (0x3e8).
Device ID This is an identification string of the device which is freely eligible by the manufacturer, fixed and unique for every device. This has the same value as defined in Sercos IDN S-0-1300.x.05.
The maximum number of characters in this string is 256.
Example: CIFX_RE_S3S_VARCFG
