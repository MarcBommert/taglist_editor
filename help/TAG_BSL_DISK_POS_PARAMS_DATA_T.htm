Disk Position

If a serial flash disk or RAM disk is used, this tag can be used to override the builtin offset and size defaults. If a parallel flash is used, this tag is ignored.

Offset The offset of the disk from the beginning of the device (serial flash or SDRAM) in bytes. The value 0xFFFFFFFF means that the default setting should be used.
Size The size of the disk in bytes. The value 0xFFFFFFFF means that the default setting should be used.
