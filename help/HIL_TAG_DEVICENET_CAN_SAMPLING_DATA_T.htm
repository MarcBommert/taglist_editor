CAN Sample Point

The sample point location is defined as >= 80% of the bit timing.
With this option it is possible to shift the sample point from the recommended default setting to an alternative setting.
This may solve timing issue when devices are used, which run out of specification regarding to the clock timings.
Note: Using this setting may make the product non-compliant to the specification!
