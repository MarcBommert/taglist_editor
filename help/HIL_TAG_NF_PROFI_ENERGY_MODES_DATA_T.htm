PROFIenergy Support

This tag allows you to enable or disable the PROFIenergy feature and to set the number of supported PROFIenergy modes.

PROFIenergy support Activation of PROFIenergy feature and number of supported PROFIenergy modes.
Allowed values:
0 == PROFIenergy is disabled.
1..8 == PROFIenergy is supported with 1..8 modes.
