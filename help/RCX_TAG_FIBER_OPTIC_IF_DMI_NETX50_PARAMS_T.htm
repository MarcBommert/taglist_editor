Fiber Optic DMI (netX50)

Fiber Optic Transceiver Diagnostic and Monitoring Interface (DMI) settings for netX50 designs. This tag defines the assignment of the I2C lines to MMIO pins.

I2C SDA1 Pin Number MMIO pin number for I2C SDA line to access DMI of the first Fiber Optic Transceiver
I2C SDA2 Pin Number MMIO pin number for I2C SDA line to access DMI of the second Fiber Optic Transceiver
I2C SCL Pin Number MMIO pin number for I2C SCL line to access DMI of the Fiber Optic Transceiver
