Hardware Data

This tag is used on devices without security memory to supply information about the hardware in order to allow firmware validation. If a security memory is found, this tag will be ignored. For a deteailed description of the fields, see the section about the "System Information Block" in the netX Dual-Port Memory Interface Manual.

Enable Enable/disable the hardware data block
Manufacturer Manufacturer code
Device Number Device number identifying the device
Serial Number The unique serial number of the device
Production Week/Year Production date: week and year minus 2000
Device Class Device Class
HW Revision Hardware revision number
HW Compatibility Hardware compatibility index
HW Option 0-3 Hardware assembly options describing the physical interface connected to each xC unit
