netX Ethernet and Fiber Optic Interface

This tag defines the the active Ethernet port(s) of netX and whether the fiber optic interface is enabled.

Selection of Active Ports Select a specific active Ethernet port for one port devices or both ports for two port devices.
  • Port 0 only
  • Port 1 only
  • Port 0 and Port 1
Enable Fiber Optic Interface support Enable Fiber Optic Mode for port 0, 1 or both.
  • Off
  • Port 0
  • Port 1
  • Port 0 and Port 1
Note: Not every firmware supports enabling Fiber Optic support for port 0 or 1 individually.
