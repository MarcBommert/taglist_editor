PROFINET IO-Link Profile Pin4 DIO in IOLM Submodule

This tag specifies where Pin4 DIO data is located. It allows you to activate handling of Pin4 DIO data in IOLM Submodule. Profile DI/DO Submodules do not have any IO data in this case.

Location of Pin4 DIO Data Specify the location of Pin4 DIO data.
Allowed values:
0: Pin4 DIO data in regular profile DI/DO submodules (default)
1: Pin4 DIO data in IOLM Submodule
