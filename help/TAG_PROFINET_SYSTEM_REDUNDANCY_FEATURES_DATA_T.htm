Profinet SystemRedundancy

In this tag, you can enable, disable or modify PROFINET SystemRedundancy related features.
Note: Please refer to PROFINET IO Device API Manual for more details about the consequence of changing this value.

Number of parallel ARSets Allowed values are 0 - 1.
0: no SR-ARSet is allowed. SystemRedundancy is not available.
1: one SR-ARSet is allowed. SystemRedundancy is available.
Please refer to technical data in PROFINET IO Device API Manual for the capabilities of the stack version used.
