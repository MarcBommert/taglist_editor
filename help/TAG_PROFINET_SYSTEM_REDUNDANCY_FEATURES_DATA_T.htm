Profinet SystemRedundancy

In this tag, you can enable, disable or modify PROFINET SystemRedundancy related features.
Note: Please refer to PROFINET IO Device API Manual for more details about the consequence of changing one of these values.

Number of parallel ARSets 0: No SR-ARSet is allowed. SystemRedundancy is not available.
1: One SR-ARSet is allowed. SystemRedundancy is available.
Allowed values are 0 - 1, please refer to technical data in PROFINET IO Device API Manual for capabilities of the stack version used.
RedundancyDataHoldTime Value 3 to 199: The minimum RedundancyDataHoldTimer according to GSDML Keyword MinRDHT that shall be accepted by the firmware.
Value 200: GSDML does not state MinRDHT
Allowed values are 3 - 200, please refer to technical data in PROFINET IO Device API Manual for capabilities of the stack version used (it may support only larger values than 3).
