SDRAM Parameters

SDRAM parameters can be set using this tag.

Note:
There are multiple locations in which SDRAM parameters may be stored. If more than one of them contains parameters, the selection depends on the 2nd Stage Loader. The current versions (1.4.x) check the the following locations, and use the first set of parameters found:

  1. The security memory
  2. The SDRAM tag of the 2nd Stage Loader (this tag)
  3. The boot header of the firmware

General Control SDRAM_GENERAL_CTRL register value.
Set to 0 if no SDRAM Parameters are available or the parameters from security memory or .NXF file should be used.
Timing Control SDRAM_TIMING_CTRL register value.
