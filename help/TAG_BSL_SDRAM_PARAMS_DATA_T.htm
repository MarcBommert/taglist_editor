SDRAM Parameters

SDRAM parameters can be set using this tag. If no parameters are given the 2nd Stage Loader will use the data provided from security memory (alternatively from the NXF Header of the firmware).

General Control SDRAM_GENERAL_CTRL register value.
Set to 0 if no SDRAM Parameters are available or the parameters from security memory or .NXF file should be used.
Timing Control SDRAM_TIMING_CTRL register value.
