UART Settings

UART number 0 or 1
Baud rate 300, 600, 1200, 2400, 4800, 9600, 19200, 38400, 57600, 115200
Parity No parity checking, odd parity, or even parity
Stop bits The number of stop bits (1 or 2)
Data bits The number of data bits (5 .. 9)
Rx FIFO trigger level Configures the receive FIFO and the "Rx FIFO ready" IRQ trigger level.
  • 0: Disables FIFO and forces immediate notification for each character received.
  • 1 .. 14: Enables FIFO. An IRQ is triggered when the receive FIFO fill level is greater than or equal to the value.
Recommended value: 8
Tx FIFO trigger level Configures the transmit FIFO and the "Tx FIFO empty" IRQ trigger level.
  • 0: Disables FIFO and forces immediate sending of each character.
  • 1 .. 14: Enables FIFO. An IRQ is triggered when the number of characters left in the FIFO is less than the value.
Recommended value: 4
RTS Mode
  • None: RTS is not used.
  • Auto/Bits: RTS is controlled by the driver, RTS forerun/trail are given in bit times.
  • Auto/Cycles: RTS is controlled by the driver, RTS forerun/trail are given in clock cycles.
  • Self: RTS is controlled by the application.
RTS Polarity
  • Default
  • Active High
  • Active Low
RTS Forerun If RTS Mode is set to Auto/Bits or Auto/Cycles, this value defines the delay between the assertion of RTS and the beginning of the start bit.
The unit of this value is either bit times or clock cycles, depending on the value of RTS mode.
Recommended values are 1 .. 5 bit times or 1 .. 10 clock cycles.
See netX Technical Reference Guide, Electrical Specifications, UART.
RTS Trail If RTS Mode is set to Auto/Bits or Auto/Cycles, this value defines the delay between the end of the stop bit(s) and de-assertion of RTS.
The unit of this value is either bit times or clock cycles, depending on the value of RTS mode.
Recommended values are 1 .. 5 bit times or 1 .. 10 clock cycles.
See netX Technical Reference Guide, Electrical Specifications, UART.
CTS Mode
  • None: CTS is not used.
  • Auto: CTS is controlled by the driver.
  • Self: CTS is controlled by the application.
CTS Polarity
  • Default
  • Active High
  • Active Low
