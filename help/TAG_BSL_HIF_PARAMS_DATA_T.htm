netX 50/100/500 HIF/DPM Parameters

DPM parameters can be set using this tag. If no parameters are given the 2nd Stage Loader will use the default settings.

Bus Type Function
Auto The bus type is determined from the load medium (PCI/DPM):
  • PCI if DeviceClass is cifX and PCI Vendor ID != 0
  • ISA if DeviceClass is cifX and PCI Vendor ID == 0
  • DPM otherwise
    Note: If DPM mode has been auto-detected, it is configured using built-in parameters. The parameters under "DPM settings" are ignored.
DPM Configures DPM mode using the parameters under "DPM settings".
ISA Configures ISA mode using built-in parameters.
PCI Configures PCI mode using the enable pin settings under "PCI settings".
Disable Disables the external bus completely.

Settings for DPM mode

The following settings are valid if the bus type is set to DPM:

IF_CONF0
IF_CONF1
IO_REGMODE0
IO_REGMODE1 		Values of the HIF timing registers.
Set the registers to 0 to use the defaults.

Settings for PCI mode

The following settings are valid if the bus type is set to PCI or if PCI mode has been auto-detected:

Use PCI enable pin Use PCI enable pin (e.g. netPLC)
Type of enable pin Type of enable pin: GPIO, PIO or HIF PIO
"Ignore" is equivalent to disabling the pin.
Inverted Indicates if the PCI enable pin is inverted.
Pin Number Pin Number of the PCI enable pin.
