netX 50/100/500 HIF/DPM Parameters

DPM parameters can be set using this tag. If no parameters are given the 2nd Stage Loader will use the default settings.

Bus Type The external bus type: Auto, DPM, ISA, PCI or Disabled.

Settings for DPM/ISA/Auto

The following settings are valid if the bus type is set to DPM, ISA or Auto:

IF_CONF0
IF_CONF1
IO_REGMODE0
IO_REGMODE1 		Values of the HIF timing registers.
These values are only required for bus types DPM, ISA (and auto). Set the registers to 0 to use the defaults.

Settings for PCI/Auto

The following settings are valid if the bus type is set to PCI or Auto:

Use PCI enable pin Use PCI enable pin (e.g. netPLC)
Type of enable pin Type of enable pin.
"Ignore" means ignore the pin and poll once.
Inverted Indicates if the PCI enable pin is inverted.
Pin Number Pin Number of the PCI enable pin.
