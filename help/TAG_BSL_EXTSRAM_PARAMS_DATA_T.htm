External Chipselect Configuration (EXT_SRAM)

Using this tag, it is possible to have the bootloader initialize the external netX chip selects (CS0-CS3). Each value has the form 0w0x0yzz according to the following table:

w 0 = 8 bit
1 = 16 bit
2 = 32 bit		 data path width
x 0-3 additional wait states after access
y 0-3 additional wait states for setup time
nCS, Aext to nOE,nWE
zz 00-3f wait states

If the register value is set to 0, the corresponding register is not touched.

MEM_SRAM0_CTRL
MEM_SRAM1_CTRL
MEM_SRAM2_CTRL
MEM_SRAM3_CTRL 		Register values to set.
