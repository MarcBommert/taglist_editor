EtherCAT Slave Product Information

The following parameters define an EtherCAT Slave device identification.

Vendor ID Vendor identification number of the manufacturer, which has been assigned to the vendor by the ETG.
All Hilscher products use the value 0xE0000044.
Product Code This is an identification number of the device which is freely eligible by the manufacturer and fixed and unique for every device.
Revision Number A number to specify the revision of the device.
