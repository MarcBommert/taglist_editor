PROFINET Product Information

The following parameters define a ProfiNet device.

Vendor ID Vendor identification number of the manufacturer, which has been assigned to the vendor by the PROFIBUS Nutzerorganisation e. V.
The valid range is 1 to 65279 (0xFEFF).
All Hilscher products use the value 0x011E.
Device ID This is a fixed and unique identification number for every device. It is assigned by the manufacturer.
The valid range is 1 to 65535 (0xFFFF).
