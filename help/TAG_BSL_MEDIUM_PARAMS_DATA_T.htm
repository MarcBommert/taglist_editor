Destination Media for Flashing

If enabled, the 2nd Stage Loader will be flashed if it was loaded via DPM/PCI or serial console, so it will always be available during startup. It is possible to override the default volume being used.

Flash Bootloader Enable/disable automatic flashing
Destination Destination device/volume for flashing.
If autodetection is selected, the following rules are followed:
  • If the BSL is running from serial flash, use serial flash as volume.
  • If the BSL was loaded via PCI/DPM and not flashed, use a RAM disk.
  • If the BSL was loaded from parallel flash, don't use any file system, but use an NXF file at 0xc0040000.
