EtherCAT Slave Mailbox Size

This tag configures the EtherCAT mailbox size.

Note: The available process data size for I/O data will be lower when a mailbox larger than 128 bytes is selected.

Mailbox Size Available options are:
  • 128 bytes RX mailbox / 128 bytes TX mailbox (default)
  • 268 bytes RX mailbox / 268 bytes TX mailbox
  • 332 bytes RX mailbox / 332 bytes TX mailbox
  • 396 bytes RX mailbox / 396 bytes TX mailbox
  • 524 bytes RX mailbox / 524 bytes TX mailbox
