PIO configuration

  • tIdentifier: name/instance of the PIO configuration
  • tMode: Configure PIO pins as inputs or outputs. Each 1 bit will configure the corresponding PIO pin as an output, each 0 as an input.
  • tDirection: Not unused, set to 0.
  • tSet: PIO Register to set PIOs
  • tClear: PIO Register to clear PIOs
  • tInput: Register to get current input value of the PIOs

tMode/tDirection/tSet/tClear/tInput

These structures define how which registers to read/write in order to operate a set of PIOs.
  • usType: value type, determines whether ulValue is simply stored to ulReg, or combined with the current value via AND or OR. In case of tInput, the value controls how the value read from the PIO register is processed.
  • ulReg: register address
  • ulValue: register value
