Web server (HTTP) configuration

This tag sets the number of the TCP port that the web server listens on.

Web server (HTTP) Port Number The TCP port number used by the web server for HTTP.

The default value is 80.
Range: 0 - 65535

If the port number is set to 0, the web server will deactivate the HTTP port.

Note: Changing the port to a number that is already used by another component inside the firmware,
will lead to undefined firmware behavior and must be avoided (check the firmware's data sheet for already used TCP ports).
