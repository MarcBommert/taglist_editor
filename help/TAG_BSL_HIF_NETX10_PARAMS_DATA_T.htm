HIF/DPM Parameters

DPM parameters for netX 10 can be set using this tag. If no parameters are given the 2nd Stage Loader will use the default settings.

Bus Type The external bus type: Auto, DPM, ISA, PCI or Disabled

Devices based on netX 10 can use either the host interface or SDRAM. By default, the host interface is enabled and the use of SDRAM is disabled. Bus Type controls this behavior. In order to activate the netX 10 SDRAM interface, set Bus Type (i.e. the host interface type) to Disabled.
