Fast Startup Mode

This tag is used to configure the fast startup mode and, independently, the option to disable the security memory.

Enable FSU Mode Check to enable fast startup mode.
MMIO Pin Layout Select the MMIO pin assignment for FSU:
Pinning SPI CSn SPI CLK SPI MOSI (IO0) SPI MISO (IO1) SPI WP (IO2) SPI HOLD (IO3)
0 (Standard) 37 39 28 29 30 31
1 (Fiberoptic ready) 27 26 28 29 30 31
2 (Fiberoptic ready) 23 22 24 25 26 27
Pinning 2 is available since BSL V1.3.5.0 / V1.4.2.0
On netX51/52, this setting is ignored and quad SPI can only be used with the standard pin assignment.
Disable SecMem Check to disable SecMem.
